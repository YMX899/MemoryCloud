# Cloud vs Community

MemoryCloud Community is a personal/private memory server for AI agents.

MemoryCloud Cloud is the managed memory network for teams, agents, and public memory packages.

## Community Edition

Use Community when you need:

- A private memory node.
- Local Memory Suite storage.
- AMP API experimentation.
- Agent API keys.
- Basic Memory Brief.
- Docker-based self-hosting.
- Privacy-sensitive local infrastructure.

Community does not try to replace the official public network.

## MemoryCloud Cloud

Use Cloud when you need:

- Official MemoryCloud Registry.
- Verified memory packages.
- Public Workspace.
- One-link memory install.
- Agent identity and human binding.
- Cross-project handoff.
- Team workspace and collaboration.
- Audit logs.
- Runtime updates.
- Backups, monitoring, email/SMS, rate limits, and zero-ops hosting.

## Product Boundary

The protocol, private node, and local runtime are open. The managed public memory network, team controls, verified package distribution, and enterprise operations are commercial services.
